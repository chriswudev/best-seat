# Best Seat

## About

Rails app for finding best seats available

### Used Libraries & Tools

- Ruby version: 3.0.2
- React on Rails
- Material UI

### Features

- CRUD for seats and movies
- Find best seat based on seat availability

### Setup project

- `bundle install`
- `rake db:create`
- `rake db:migrate`
- `yarn`

### Run project

- `rails s`
